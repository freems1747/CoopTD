## 🎮 주요 기능

### ⚔️ 강화 시스템
- LV **0 → 20** 강화
- 레벨별 **개별 강화 확률**
- 실패 시 **확률 +1% 누적 (최대 +5%)**
- 성공 시 해당 구간 실패 보정 초기화

---

### 💎 재화 시스템
- 강화 시 **해방석 소모**
- 누적 소모량 실시간 표시

---

### 🤖 자동 강화
- 목표 강화 수치 입력
- **0.5초 간격 자동 강화**
- 자동 강화 중:
  - 수동 강화 ❌
  - 버튼이 **[중지]** 로 변경
- 언제든지 중지 가능

---

### 📊 통계 & 로그
- 강화 로그 (성공 / 실패 색상 구분)
- **강화 구간별 시도 횟수**
